### 🎛 So... How to set up the environment?

Install dependencies and then run `start` to develop locally or run `build` to create a build for production

```bash
# Installing dependencies
npm install
# or
yarn

# Running locally
npm start
# or
yarn start

# Build for production
npm run build
# or
yarn run build
```
